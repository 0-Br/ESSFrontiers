### 课程论文

**地球系统的概率流形理论: 微分几何视角下的随机动力学过程**

从微分几何的视角出发，将地球系统描述为流形上的随机动力学过程。提出物理约束将系统状态限制在低维光滑流形上（流形假设），系统演化遵循流形上的 Itô 随机微分方程，并由 Fokker-Planck 方程导出不变概率测度。通过初值不确定性传播导出度量张量，自然编码系统的可预报性极限。基于流形吸引子的拓扑结构，将气候变化分类为测度平滑演化（I 类）与吸引子拓扑突变（II 类），并讨论了扩散模型在数据驱动流形学习中的应用。
